## Encrypt password by openssl

If you want to automate shell scripts such as ssh connection and sudo, you must use your passwords.

However, when password is written in the scripts, everyone can read the passwords.

So, you encrypt the passwords not to read around other peaple.

To encrypt a password for example 'hogehoge', below to type and run words.

``
openssl genrsa -out password.key 2048
echo 'hogehoge' | openssl rsautl -encrypt -inkey password.key > password.txt
ls
``
